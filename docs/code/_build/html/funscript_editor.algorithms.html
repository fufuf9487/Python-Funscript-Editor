Bases: object

Video tracker for static feature tracking. We assume that the tracked feature is always in the neighborhood fo the first tracking box. This allows an accelerated prediction.

Args:

first_frame (np.ndarray): open cv image representing the start frame tracking_bbox (tuple): tuple with (x,y,w,h) of the init tracking box limit_searchspace (bool) : only insert the region of init tracking box into the tracker queue_size (int): in (work) and out (result) queue size
